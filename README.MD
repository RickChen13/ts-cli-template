# ts-cli-remplate

## 依赖安装

~~~bash
yarn
~~~

## 运行

~~~bash
yarn dev:main 
# will run src/main.ts 
# [For details, you can go to package.json to see "script"]

# or 
yarn dev your-need-run-script-path
~~~

## 编译

~~~bash
yarn build 
# will build src/main.ts 
# [For details, you can go to package.json to see "Tsup"]
~~~
